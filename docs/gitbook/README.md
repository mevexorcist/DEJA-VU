# DEJA-VU

## Crypto Social Platform

DEJA-VU adalah platform media sosial untuk komunitas cryptocurrency dengan fitur airdrop farming dan integrasi exchange.

### Visi

Membangun platform sosial Web3 yang menghubungkan komunitas crypto, memudahkan discovery airdrop, dan menyediakan tools untuk portfolio tracking.

### Fitur Utama

- **Social Feed** - Timeline seperti X.com untuk berbagi update crypto
- **Airdrop Discovery** - Temukan dan track airdrop terbaru
- **Portfolio Tracking** - Monitor holdings dan performa investasi
- **Direct Messages** - Komunikasi privat antar pengguna
- **Notifications** - Real-time alerts untuk aktivitas penting

### Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS dengan design system X.com + Linear.app
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Deployment**: Docker, Nginx
