# Features

## Social Feed

### Home Timeline

Feed utama menampilkan posts dari pengguna yang kamu follow dan trending content.

**Fitur:**
- Infinite scroll
- Like dan repost
- Reply threads
- Media attachments (images, videos)
- Crypto price mentions auto-link

- Hashtags (#DeFi, #NFT)
- Mentions (@username)
